Bem vindo(a)!


Esta aplicação foi feita para interagir e testar códigos SQL nos Sistemas de Gerenciamento de Banco de Dados (SGDBs) PostgreSQL e MySQL





Como usar


Para utilizar esta aplicação, basta seguir os passos a seguir:

  1. Passe o mouse por cima da barra lateral à esquerda da página.

  2. Clique no item do SGDB que desejar: MySQL ou PostgreSQL.
    Caso queira mais informações sobre o projeto de TCC como um todo clique em Sobre.

  3. Ao abrir a nova página, escreva seu código SQL na caixa de texto e clique no botão Executar Query embaixo da própria caixa de texto, o resultado será mostrado abaixo desse botão.
    Para limpar o resultado devolvido clique no botão Limpar Resultado.

  4. Caso queira utilizar a UI gráfica, ela se localiza na caixa à direita da caixa de texto.
    Para usá-la, basta preencher o formulário e clicar no botão Submit para inserir os dados, Patch para alterar os dados ou Delete para excluir os dados.
    Clicando em GetDB o banco de dados inteiro será exibido no campo de resultados abaixo do botão Executar Query.

  5. Importante: A UI gráfica para o banco de dados inicial exige o preenchimento do campo Email, que é a chave primária, para poder realizar as inserções, alterações e exclusões.
